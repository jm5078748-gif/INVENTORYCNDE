Langkah Setup:

1. Buka Google Spreadsheet Anda

2. Buat sheet bernama Transaksi dan MasterMaterial

3. Klik Extensions → Apps Script

4. Hapus kode default, paste kode GAS dari file gas-script.gs

5. Klik Deploy → New Deployment → Web App

6. Set Execute as: Me, Who has access: Anyone

7. Copy URL Web App, paste di kolom di atas

8. Di sheet MasterMaterial, buat header di baris 1:

A1: KodeMaterial   B1: Deskripsi   C1: Satuan
